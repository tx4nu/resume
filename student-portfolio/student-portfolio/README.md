# 🎓 Student Portfolio & Git Version Control Lab

This repository contains a responsive Student Portfolio website built as part of **Experiment No. 2: Version Control using Git**.

## 📌 Project Features
- **Student Profile**: Name, specialization, and short bio.
- **Projects Section**: List of key student projects.
- **Contact Details**: Email and GitHub profile.
- **Responsive Styling**: Custom CSS layout (`style.css`).

## 🛠️ Version Control Steps Executed
1. Initialized repository (`git init`).
2. Staged and committed initial files (`git commit -m "Create student portfolio homepage"`).
3. Updated `index.html` with email info.
4. Staged and committed updates (`git commit -m "Add email information to portfolio"`).
